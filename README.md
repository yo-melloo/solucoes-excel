# Soluções Excel - Por Gustavo Mello
Este repositório reúne planilhas em Excel desenvolvidas para otimizar o controle operacional do setor de tráfego de uma empresa de transporte rodoviário.
Os arquivos foram estruturados para oferecer eficiência, automatização e clareza nas rotinas diárias de gestão de frotas, escalas e motoristas.

## Conteúdo do Repositório

### 1. Sistema de Gerenciamento de Tráfego (SISGET - Dashboard Geral de Tráfego)

Planilha consolidada que integra informações de escala de motoristas, disponibilidade de veículos e controle de frotas.

- Principais recursos:
    - Visão geral das operações diárias.
    - Banco de dados para geração de bafômetros automátizados
    - Painel dinâmico com indicadores automáticos.
    - Controle de escalas e substituições.

### 2. Controle de Refeições (IMP e STI)

Planilhas destinadas ao monitoramento de refeições em duas bases distintas (IMP e STI), com automações integradas para facilitar o acompanhamento.

- Principais recursos:
    - Sistema automático de contagem de refeições e identificação dos motoristas.
    - Interface simples e padronizada para cada base.
    - Relatórios consolidados e exportação rápida.


### 3. Documento de Folhas de Rosto (Bafômetros)

Arquivo utilizado para a geração de documentos oficiais de controle de testes de alcoolemia de motoristas.

- Principais recursos:
    - Modelo de folha de rosto automatizado.
    - Identificação e por escala.

### Objetivo

Centralizar e padronizar os controles administrativos do setor de tráfego, garantindo maior precisão nos registros e eficiência nos processos operacionais.
