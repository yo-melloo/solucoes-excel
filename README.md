# 📊 Soluções Excel - Satélite Norte (Imperatriz)

Repositório central de inteligência operacional e gestão de ativos da filial Satélite Norte em Imperatriz/MA. Este espaço é dedicado à organização de planilhas complexas, documentos técnicos e uma base de conhecimento estruturada via Obsidian.

## 🗂️ Estrutura do Repositório

- **`Satélite Norte/`**: Arquivos operacionais (Excel, Word, PDF) para controle de frota, refeições e bafômetros.
- **`sisget/`**: (Ignorado/Repo Independente) Sistema de Gerenciamento de Tráfego Web.
- **`.obsidian/`**: Configurações da base de conhecimento (Zettelkasten).

## 🧠 Base de Conhecimento (Obsidian)

Este repositório foi projetado para ser aberto como um **Vault do Obsidian**. Ele centraliza:
- Notas técnicas de processos da filial.
- Documentação de fluxos de tráfego.
- Referências para medição de tanques e bafômetros.

## 🛠️ Como Utilizar

1. **Gestão de Planilhas**: Localize os arquivos em `Satélite Norte/`. Mantenha o padrão de nomes para garantir a compatibilidade com os scripts de extração de dados.
2. **Documentação**: Abra a raiz deste repositório no Obsidian para visualizar o grafo de conhecimento e documentos interligados.
